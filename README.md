# Handwritten_Digit_Recognition
📌 Project Description

This project implements a Handwritten Digit Recognition system using a Convolutional Neural Network (CNN). The model is trained on the MNIST, which contains thousands of labeled images of handwritten digits from 0 to 9.

The goal of this project is to train a deep learning model that can accurately recognize handwritten numbers from image data.

🎯 Objectives

To build a CNN model for digit classification.

To train the model using the MNIST dataset.

To achieve high accuracy in recognizing handwritten digits.

To visualize model performance using graphs and a confusion matrix.

🛠 Technologies Used

Python

TensorFlow

Keras

NumPy

Matplotlib

Scikit-learn

Seaborn

📂 Dataset

The project uses the MNIST dataset, which contains:

60,000 training images

10,000 testing images

Image size: 28 × 28 pixels

Grayscale format

Each image represents a digit from 0–9.

⚙️ Project Workflow

Import required libraries.

Load the MNIST dataset.

Preprocess the dataset (normalization and reshaping).

Build the CNN model.

Compile and train the model.

Evaluate model accuracy.

Visualize results using graphs and confusion matrix.

Predict digits from random test images.

🧠 CNN Architecture

The CNN model consists of the following layers:

Convolution Layer (Conv2D)

Activation Function (ReLU)

Max Pooling Layer

Flatten Layer

Fully Connected Dense Layer

Softmax Output Layer

These layers help the model extract features from images and classify digits correctly.

📊 Results

Test Accuracy achieved: ~98%

The model successfully predicts handwritten digits from unseen images.

Confusion matrix shows strong classification performance across digits.
